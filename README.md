## Description:
The "Web Development" application presents the basic issues needed to create websites.
It has been divided into modules:
- Terminal
- HTML Tags
- HTML Inputs
- HTML Sections
- CSS Positioning
- CSS Flex
- CSS Grid
- JS Numbers
- JS Strings
- JS Arrays
- JS Objects

Modules contain a description of a given topic or present examples of applications.
